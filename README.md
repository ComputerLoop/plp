Programing lenguage plp
=========================

**Plp** is a *programming language* that contains basic functions

varsion: alfa-2.0

What's new
-------

- function
- better error messages

Syntax
-----------

```hs
% comment

test: 3 + 4 - 5;
puts test; % output 2

function (
  var_x: 4;
  puts var_x; % 4
)

!function; % call function

puts var_x;
```

Run Program
-----------

```ps1
plp "path/to/file.plp"
```
