Programing lenguage plp
=========================

Plp - a simple language that contains basic functions

Syntax
-----------

foo **:** 4 **;** *% this comment*

baz **:** 3 **-** foo **+** 4;

***puts*** baz; *% output*

Run Program
-----------

```ps1
plp "path/to/file.plp"
```
